 Добрый день!

 Это тестовое задание для компании NewGen Vision.
Подробное описание задания находится по ссылке https://newgen-it.github.io/tests/front.

 Как отфильтровать список курсов, чтобы выдались только подходящие по цене?
 Я поняла данное задание так:
 1. Есть список курсов, у которых есть массив цен состоящий из 2 элементов, которые создают диапозон
цен курса;
 2. Есть фильтр, который содержит второй диапозон цен;
 3. В некоторых местах вместо чисел указано "null", значит с этой стороны диапозон не ограничен, то есть уходит в
бесконечность.

 Исходя из пунктов, указанных выше, сделала функцию, которая сначала фильтрует массив по входным данным, потом
сортирует его по возврастанию начальной цены.
 Фильтрация массива ищет пересечения диапозонов цен, то есть если пользователь ищет курс стоимостью от 100 до 350, то
функция возвращает все массивы, в которые входит этот отрезок (например, от 100 до 200). В том случае если один из
элементов запроса или элементов цен списка курсов равен "null", я приравниваю данное значение к бесконечности или
- бесконечности, в зависимости от расположения элемента.
 Сортировка выполнена пузырьковой, то есть сравниваются 2 элемента массива, если первый больше второго то они меняются
местами, такой цикл выполняется к каждому элементу массива. После первого прохождения по массиву на последней позиции
находится самый большой элемент, далее цикл повторяется, но не затрагивает уже отсортированный последний элемент,
повторяется до тех пор пока весь массив не будет отсортирван.
